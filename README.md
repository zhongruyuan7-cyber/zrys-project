## 🎨 数字化视觉设计说明
本项目完全摒弃了传统的单调网页排版，全栈采用现代前沿 UI 设计语言——**玻璃拟态（Glassmorphism）与暗色炫彩霓虹流**。针对每一周的独特核心课题，定制专属的霓虹唤醒色（璀璨金、芯片蓝、魅惑紫、极客红等），打造具有极客品牌感的个人数字化作品集。
